1. Follow instuctions on https://ionicframework.com/getting-started to get node.js and install ionic

2. Clone the repository from https://github.com/webbryan92/ionPickBan

3. Run npm install to install modules from package.json

4, Run ionic serve in the directory, then nagivate to localhost:8100 to see the project

5. If its still not working run "ionic state restore" in the project directory.
