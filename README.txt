1. 	Follow instuctions on https://ionicframework.com/getting-started to get node.js and install ionic

2. 	Clone the repository from https://github.com/webbryan92/ionPickBan

3. 	Run ionic serve in the cloned directory, select yes to install project dependancies(runs npm install)
	then nagivate to localhost:8100 in a web browser to see the project

4. 	If its still not working run "ionic state restore" in the project directory.
